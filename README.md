# react-native-easy-layout

一个类似flutter布局的react-native布局组件库

## 安装
工程目录下运行：<br>
 ```bash
 npm install --save react-native-easy-layout
```
or(已经安装了yarn)<br>
```bash
 yarn add react-native-easy-layout
```
